# 2. Hero-komponent i Next.js + Tailwind

Här är ett exempel på en `HeroSection`-komponent som ni kan lägga i t.ex. `src/components/HeroSection.tsx` (eller liknande).

* Mörk bakgrund
* “Plasma”-liknande gradient som rör sig i bakgrunden
* H1 + underrad
* Två knappar med scroll till sektioner (`#contact` och `#services`)
* Liten trygghetstext

# 1. Textinnehåll

## Sektionstitel:
Berätta om ditt projekt

## Introtext:
Fyll i några korta rader om vad du vill bygga, så återkommer vi med frågor, förslag eller en offert.

## Fält i formuläret:

* **Namn / kontaktperson** (obligatoriskt)
* **E-post** (obligatoriskt)
* **Företag / organisation** (valfritt)
* **Typ av projekt (select)**
    * Webbsida / webbapp
    * Mobilapp (iOS / Android)
    * Skräddarsytt system / integration
    * Annat / osäker
* **Budgetnivå (select, valfritt):**
    * < 20 000 kr
    * 20 000 – 50 000 kr
    * 50 000 – 100 000 kr
    * 100 000+ kr
    * Vet inte än
* **Meddelande / beskrivning** (textarea)
* **Liten text:** “Vi återkommer normalt inom 1–2 arbetsdagar.”
